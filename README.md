# Student Grade Tracker

## 📌 Project Description
Student Grade Tracker is a Java console-based application developed to manage student grades efficiently. The application allows users to enter student names and marks, stores the data using an ArrayList, calculates the average, highest, and lowest marks, and displays a well-formatted summary report.

## ✨ Features
- Add student names and marks
- Store student data using ArrayList
- Calculate average marks
- Find highest marks
- Find lowest marks
- Display a formatted student report
- Validate marks (0–100)

## 🛠️ Technologies Used
- Java
- VS Code
- JDK
- ArrayList
- Object-Oriented Programming (OOP)

## ▶️ How to Run
1. Open the project in VS Code.
2. Compile the program:
   ```
   javac StudentGradeTracker.java
   ```
3. Run the program:
   ```
   java StudentGradeTracker
   ```
4. Enter the student details when prompted.
